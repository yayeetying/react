Intro to React

  • Javascript library
  • build website's user interface
  • open-source, free, and popular!
  • maintained by Facebook; used by Instagram, Discord, Netflix

React is like an HTML rendering engine

write Javascript to describe your page's HTML, and React changes the page's DOM accordingly


pros cons
  • no flipping between HTML and JS files
  • allows for modular code through "components"
  • don't need to manually change DOM
  • learning curve for developer
  • extra abstraction (React renders HTML) --> slower
  • React ~38kb library
